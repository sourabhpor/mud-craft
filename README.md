# 🧶 Mud Craft – Full Stack E-commerce Application

A full-stack e-commerce application built using modern technologies with separate frontend and backend architecture.

---

## 📁 Project Structure

```
mud-craft/
│
├── frontend/        # React / vite Application
├── backend/         # NestJS / Node.js API
├── README.md
```

---

## 🚀 Tech Stack

### 🔹 Frontend

- React.js
- Axios
- Tailwind CSS / CSS

### 🔹 Backend

- NestJS
- TypeORM
- PostgreSQL
- JWT Authentication

---

## ⚙️ Prerequisites

Make sure you have installed:

- Node.js (>= 18)
- npm or yarn
- PostgreSQL

---

# 🖥️ Running the Project

---

## 🔹 1. Clone Repository

```
git clone <your-repo-url>
cd mud-craft
```

---

## 🔹 2. Backend Setup

```
cd server
npm install
```

### ▶️ Run Backend (Development)

```
npm run start:dev
```

### ▶️ Run Backend (Production)

```
npm run build
npm run start:prod
```

---

## 🔹 3. Frontend Setup

```
cd client
npm install
```

### ▶️ Run Frontend

```
npm start
```

```
npm run dev
```

---

## 🔐 Environment Variables

Create `.env` file inside **server/.**

---

## 🧪 Running Tests (Backend)

```
cd backend
npm test
```

---

## 📦 Build

### Backend

```
cd server
npm run build
```

### Frontend

```
cd client
npm run build
```
